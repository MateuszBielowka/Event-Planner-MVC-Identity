# Event Planner MVC Identity

## Overview
Event Planner MVC Identity is a web application built with ASP.NET Core MVC and Identity for secure user authentication. It enables users to efficiently manage events, employees, assignments, and expenses, streamlining the process of organizing and tracking event-related activities.

## Features
- User authentication and authorization with ASP.NET Identity
- CRUD operations for events, employees, assignments, and expenses
- Structured views for managing and overviewing event data
- Responsive and user-friendly interface

## Folder Structure
- Controllers/ - Handles business logic for events, employees, assignments, and expenses
- Models/ - Data models for application entities
- Views/ - Razor views for UI rendering
- Data/ - Application database context
- Areas/Identity/ - Identity pages for authentication
- wwwroot/ - Static files (CSS, JS, libraries)

## Getting Started
1. Clone the repository
2. Restore NuGet packages
3. Build the project using Visual Studio or msbuild
4. Run the application and access it via your browser

## Requirements
- .NET 7.0 or later
- SQL Server (local or remote)
